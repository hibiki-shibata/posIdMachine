Upload Menu JSON File

    This tool will helps to register POS ID in every single items in venues.


  1. Export Menu JSON file in old menu editor.

  2. Select the file from "Choose File" in this tools.

  3. When you hit the "Register POS ID button", vola! POS ID is registered in all items.

  4. You can import the downloaded new menu file, the name start with has_posID~.
